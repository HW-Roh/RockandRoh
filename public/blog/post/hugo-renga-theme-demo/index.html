Features

  • Minimalist
  • Responsive & mobile-first (Bootstrap 4)
  • Native font stack
  • High Contrast
  • Math typesetting (MathJaX)
  • Syntax highlighting (highlight.js)
  • Progressive image loading (progressively.js)
  • Disqus
  • Google Analytics
  • Open Graph meta tags
  • Twitter Cards meta tags
